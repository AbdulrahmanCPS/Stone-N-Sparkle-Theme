# Fonts and other assets — licensing

This theme references fonts and other assets that may have **separate licensing** from the theme code (GPL v2 or later). Redistribution rights must be verified before making this repository or any distribution public.

## Fonts referenced (see `assets/css/main.css`)

| Asset | Filename / note | Action before public release |
|-------|-----------------|------------------------------|
| **SS Mollina** | `mollina.otf` | Confirm you have redistribution rights. |
| **Parkinsans** | `assets/fonts/parkinsans/*.woff2` | Licensed under SIL Open Font License 1.1. Full license text: [assets/fonts/parkinsans/OFL.txt](assets/fonts/parkinsans/OFL.txt). See [openfontlicense.org](https://openfontlicense.org). |
| **Al Jazeera Arabic** | `Al-Jazeera-Arabic-Bold.ttf` | Confirm trademark/license allows redistribution in this theme. |

## Before going public

1. **Replace or remove** any trial/proprietary fonts, or obtain licenses that allow public redistribution.
2. **Document** in this file (or in `README.md`) the license and source for each font you ship.
3. Ensure all other bundled assets (images, icons, etc.) are either your own, GPL-compatible, or used under a license that allows redistribution.
